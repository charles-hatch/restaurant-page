## Restaurant Page

A small restaurant website built as part of [The Odin Project](https://www.theodinproject.com/) JavaScript curriculum, revisited later using **Webpack** to bundle modules and assets.

The site content is generated with JavaScript and organized into tabbed sections (e.g., Home / Menu / Contact) using ES6 modules.

### Tech

- HTML
- CSS
- JavaScript (ES6 modules)
- Webpack (bundling, dev server)
